SplInt::__construct

(PECL spl_types >= 0.1.0)

SplInt::__construct — Constructs an integer object type

Description

SplInt::__construct() ( integer $input )

This constructs a new object of type integer.

Parameters

input

The input parameter accepts an integer An UnexpectedValueException Exception will be thrown if anything is passed.

Return Values

No value is returned.

Examples

Example #1 SplInt::__construct() example

<?php
$int = new SplInt(94);

try {
    $int = 'Try to cast a string value for fun';
} catch (UnexpectedValueException $uve) {
    echo $uve->getMessage() . PHP_EOL;
}

var_dump($int);
echo $int; // Outputs 94
?>

The above example will output:

Value not an integer
object(SplInt)#1 (1) {
  ["__default"]=>
  int(94)
}
94
