SplString::__construct

(PECL spl_types >= 0.1.0)

SplString::__construct — Constructs a string object type

Description

SplString::__construct() ( string $input )

This constructs a new object of type string.

Parameters

input

The input parameter accepts a string An UnexpectedValueException Exception will be thrown if anything else is passed.

Return Values

No value is returned.

Examples

Example #1 SplString::__construct() example

<?php
$string = new SplString("Testing");

try {
    $string = array();
} catch (UnexpectedValueException $uve) {
    echo $uve->getMessage() . PHP_EOL;
}

var_dump($string);
echo $string; // Outputs "Testing"
?>

The above example will output:

Value not a string
object(SplString)#1 (1) {
  ["__default"]=>
  string(7) "Testing"
}
